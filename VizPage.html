Data Visualizations

An interactive force-directed graph layout. You can moves the nodes around, and double click on any node to pin it in place! Each node has a source name, target name, and decimal value. The size of the nodes are based on their respective degrees, with an exponential scale to the 3rd power to emphasize higher degree nodes. The edge colors are a reflection of the value of the edges:
< 1.0 = blue
1.0 ≤ edge value ≤ 2.0 = green
> 2.0 = red
Source file for the node data can be found here.
